# auto-fix-cycle - Autonomous Error Fixing

Automatically detect and fix TypeScript type errors and ESLint violations through iterative analysis and refinement.

## Safety Checks (Execute First)

Verify TypeScript project:

```bash
test -f tsconfig.json || test -f package.json
```

Check for uncommitted changes:

```bash
git status --porcelain
```

## Phase 1: Error Detection

Scan for all errors in parallel

Execute:
pnpm run lint pnpm run typecheck

Validation: Parse output to create error list with file, line, code, message

## Phase 2: Error Fixing

Fix each error autonomously

For each error in priority order:

1. Grep for exact error location 2. Read surrounding context (±20 lines) 3. Analyze and apply fix with Edit tool 4. Validate: re-run checks for that file
   Max 3 iterations

Validation: Track progress with TodoWrite

## Phase 3: Validation

Run full quality pipeline

Execute:
pnpm run quality

Validation: Count errors fixed vs remaining
