# Loop Prepare - Pre-Loop Safety Setup

Run this ONCE before starting overnight `/loop` sessions. Validates safety, creates snapshot, and sets up logging.

## Step 1: Git Safety Check

Run these in parallel:

```bash
git rev-parse --is-inside-work-tree 2>&1
```

```bash
git branch --show-current
```

```bash
git status --porcelain | head -20
```

If on `main` or `master` branch, STOP:

```
================================================================
  LOOP PREPARE - BLOCKED
================================================================

Current branch: main

Overnight loops MUST NOT run on main.
Create a loop branch first, or let this command create one.

================================================================
```

## Step 2: Check Commits Ahead

```bash
git log main..HEAD --oneline 2>/dev/null | wc -l
```

If >= 10 commits ahead of main:
1. Report: "You have N commits ahead of main. Merging first..."
2. Execute incremental merge:

```bash
git checkout main && git pull origin main 2>/dev/null; git merge @{-1} --no-ff -m "chore: incremental merge before overnight loops" && git checkout -
```

If merge fails, STOP and report conflict.

## Step 3: Create Loop Branch

Create a dated loop branch:

```bash
BRANCH_NAME="loop/overnight-$(date +%Y%m%d)"
git checkout -b "$BRANCH_NAME" 2>/dev/null || git checkout "$BRANCH_NAME"
echo "On branch: $BRANCH_NAME"
```

## Step 4: Create D:\ Snapshot

```bash
cd C:/dev/scripts/version-control && powershell -ExecutionPolicy Bypass -File Save-Snapshot.ps1 -Description "Before overnight loops $(date +%Y-%m-%d)" -Tag "pre-loop-$(date +%Y%m%d)"
```

If snapshot fails, WARN but continue (don't block on this).

## Step 5: Verify Disk Space

```bash
powershell -Command "
  $cFree = (Get-PSDrive C).Free / 1GB
  $dFree = (Get-PSDrive D).Free / 1GB
  Write-Host \"C: drive free: $([math]::Round($cFree, 1)) GB\"
  Write-Host \"D: drive free: $([math]::Round($dFree, 1)) GB\"
  if ($cFree -lt 5 -or $dFree -lt 5) {
    Write-Host 'WARNING: Less than 5GB free on a drive!'
    exit 1
  }
  Write-Host 'Disk space OK'
"
```

If < 5GB free on either drive, WARN but continue.

## Step 6: Kill Stale Processes

```bash
powershell -ExecutionPolicy Bypass -File C:/dev/scripts/cleanup-processes.ps1
```

## Step 7: Create Session Directory and State Files

```bash
SESSION_DIR="D:/logs/loop-sessions/$(date +%Y%m%d)"
mkdir -p "$SESSION_DIR"

# Initialize quality sweep state
cat > "$SESSION_DIR/quality-sweep-state.json" << 'STATEEOF'
{
  "lastProject": null,
  "projectIndex": 0,
  "commitCount": 0,
  "commitBudget": 20,
  "processedProjects": [],
  "failedProjects": [],
  "consecutivePrecommitFailures": 0,
  "startedAt": null,
  "mergeCount": 0
}
STATEEOF

# Initialize cleanup state
cat > "$SESSION_DIR/cleanup-state.json" << 'CLEANEOF'
{
  "currentPhase": 1,
  "commitCount": 0,
  "commitBudget": 10,
  "completedPhases": [],
  "startedAt": null
}
CLEANEOF

# Initialize review report
cat > "$SESSION_DIR/review-report.md" << 'REPORTEOF'
# Overnight Code Review Report

Generated by `/loop review-optimize`

| Project | Complexity | Anti-Patterns | Security | Status |
|---------|-----------|---------------|----------|--------|
REPORTEOF

# Initialize log
echo "[$(date -Iseconds)] [prepare] Session initialized" > "$SESSION_DIR/loop.log"

echo "Session directory: $SESSION_DIR"
echo "State files created."
```

## Step 8: Run Baseline Health Check

```bash
powershell -ExecutionPolicy Bypass -File C:/dev/scripts/workspace-health.ps1 2>&1 | tail -20
```

## Step 9: Print Summary

After all steps complete, print this summary:

```
================================================================
  LOOP PREPARE - READY
================================================================

Branch:     loop/overnight-YYYYMMDD
Snapshot:   pre-loop-YYYYMMDD (D:\repositories\vibetech)
Session:    D:\logs\loop-sessions\YYYYMMDD\
Disk:       C: XX.X GB free | D: XX.X GB free
Health:     PASSED

Start loops in separate terminals:

  Terminal 1:  /loop 15m loop:quality-sweep
  Terminal 2:  /loop 30m loop:cleanup-hygiene
  Terminal 3:  /loop 60m loop:review-optimize

Commit budgets: quality=20, cleanup=10, review=0 (read-only)
Auto-merge to main every 10 commits.

To rollback everything:
  cd C:\dev\scripts\version-control
  .\Restore-Snapshot.ps1 -Tag "pre-loop-YYYYMMDD"

================================================================
```

Require user to confirm before they start the loops: "Ready to start overnight loops? Start each /loop in a separate terminal."
