# initialization - Agent Orchestration Setup

Initialize specialist agent orchestration system for desktop-apps, infrastructure, quality-assurance, optimization, or web-apps domains.

## Safety Checks (Execute First)

Verify workspace structure:

```bash
test -d C:/dev/.claude/agents && test -d C:/dev/.claude/hooks
```

Check for existing orchestration:

```bash
test -f C:/dev/.claude/agents.json
```

## Phase 1: Domain Detection

Identify target domain

Execute:
if [ -z "$1" ]; then echo "Usage: /initialization [desktop-apps|infrastructure|quality-assurance|optimization|web-apps]"; exit 1; fi DOMAIN=$1

Validation: Ensure domain is one of the 5 supported types

## Phase 2: Agent Registration

Register specialist agents for domain

Execute:
grep -r "tags._$DOMAIN" C:/dev/.claude/agents/_.md | awk -F: '{print $1}'

Validation: At least 1 agent found for domain

## Phase 3: Orchestration Setup

Configure agent delegation rules

Execute:
cat C:/dev/.claude/agent-delegation.yaml

Validation: Delegation rules include domain agents

## Phase 4: Verification

Test orchestration system

Execute:
pnpm nx graph echo "Orchestration initialized for $DOMAIN domain"

Validation: No errors, agents registered
