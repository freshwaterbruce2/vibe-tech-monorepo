⚠️ **Editing pnpm-lock.yaml directly!**

This file should be generated by pnpm, not edited manually.

**Proper workflow:**
```bash
pnpm install  # Regenerate lock file
```

If you need to update dependencies:
```bash
pnpm update <package>  # Update specific package
pnpm update  # Update all packages
```
