Glass & Effects

Vibe-Tech uses a neon glassmorphism design language. Frosted glass surfaces with subtle neon accents create depth without overwhelming the content.

Glass Surfaces

Glass Primary

Default panel background. Use for cards, modals, and content areas.

Subtle, professional cards with hover states.

Usage Guidelines

Logo Usage

Do

  • Use the logo on dark backgrounds (#07090F-#12141C)
  • Maintain clear space equal to the logo's height on all sides
  • Use the full-color version as the default
  • Apply a subtle purple glow for hero placements

Don't

  • Place on light/white backgrounds without a dark container
  • Stretch, rotate, or apply non-brand filters
  • Reduce below 32px in any dimension
  • Add drop shadows or outlines not in the brand palette

Color Usage

Do

  • Use Electric Violet (#7C3AED) for primary actions and focus rings
  • Use Cyan (#00FFFF) sparingly for accent highlights and links
  • Keep neon colors at low opacity (0.1-0.4) for glows and borders
  • Pair neon accents with dark, desaturated backgrounds

Don't

  • Use neon colors as large solid fills (overwhelming)
  • Mix more than 2 neon accents in a single component
  • Use pure black (#000000) — always use the dark slate palette
  • Place light text on neon backgrounds without contrast check

Typography

Do

  • Use Montserrat for headings and display text only
  • Use Inter for all body copy, labels, and descriptions
  • Use JetBrains Mono for code, tokens, and technical values
  • Maintain heading hierarchy (Display -> H1 -> H2 -> H3)

Don't

  • Use Montserrat for body text or long paragraphs
  • Mix more than 2 font weights in a single component
  • Use font sizes below 12px for any readable content
  • Apply gradient text to body copy (headings only)

Glass & Effects

Do

  • Layer glass surfaces to create depth hierarchy
  • Use blur(12px) for standard panels, blur(16px) for nav
  • Keep glass borders at 0.06-0.10 white opacity
  • Apply neon glows to interactive elements on hover/focus

Don't

  • Stack more than 3 glass layers (performance + readability)
  • Use glass on light backgrounds (needs dark to work)
  • Apply constant glow animations to non-interactive elements
  • Use blur values above 20px (diminishing returns)
