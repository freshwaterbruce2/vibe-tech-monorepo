# App Icon Design Guide

**Required:** 1024×1024 PNG, no alpha channel

---

## 🎨 Design Options

### Option 1: Lightbulb + Graduation Cap (Recommended)

**Symbolizes:** Learning + AI intelligence

**Design:**

- Central lightbulb (represents AI/ideas)
- Graduation cap on top (represents education)
- Purple-to-cyan gradient background
- Neon glow effect

**Colors:**

- Primary: #D946EF (Hot Magenta)
- Secondary: #06FFF0 (Electric Cyan)
- Background: #0F0F23 (Dark Purple)

### Option 2: "VT" Monogram

**Symbolizes:** Vibe-Tutor brand

**Design:**

- Large "VT" letters
- Overlapping style
- Gradient fill (purple to cyan)
- Rounded, modern font
- Subtle glow

### Option 3: Brain + Tech Hybrid

**Symbolizes:** Neurodivergent support + technology

**Design:**

- Stylized brain icon
- Circuit board pattern overlay
- Gradient colors
- Modern, friendly style

---

## 🛠️ How to Create (Canva - Easiest)

**Step 1:** Go to <https://canva.com/create/app-icons/>

**Step 2:** Search templates: "education app icon" or "learning app"

**Step 3:** Customize:

- Change colors to purple (#D946EF) and cyan (#06FFF0)
- Replace text with "VT" or remove text entirely
- Add lightbulb or brain icon if template allows

**Step 4:** Download:

- File type: PNG
- Size: 1024×1024
- Quality: Highest
- Background: Solid color (no transparency)

**Step 5:** Save to `store-assets/icon-1024x1024.png`

---

## 🛠️ How to Create (Figma - More Control)

**Step 1:** Go to <https://figma.com>

**Step 2:** Create new file → Create frame 1024×1024

**Step 3:** Design elements:

### Background

- Rectangle: 1024×1024
- Fill: Linear gradient
  - Start: #D946EF (top-left)
  - End: #06FFF0 (bottom-right)
  - Angle: 135°

### Icon Element (Choose One)

**Lightbulb:**

- Search icons: "lightbulb"
- Size: ~600×600
- Color: White (#FFFFFF)
- Position: Center
- Add drop shadow: Blur 20px, Y offset 10px, opacity 50%

**Graduation Cap:**

- Search icons: "graduation cap"
- Size: ~400×400
- Position: Top-right of lightbulb
- Color: White or Yellow

### Glow Effect

- Duplicate icon layer
- Blur: 40px
- Opacity: 60%
- Color: Brighter purple/cyan

**Step 4:** Export:

- Select frame
- Export as PNG
- 1x scale (1024×1024)
- Save to `store-assets/icon-1024x1024.png`

## ✅ Icon Checklist

**Technical Requirements:**

- [ ] Size: Exactly 1024×1024 pixels
- [ ] Format: PNG (32-bit)
- [ ] No transparency/alpha channel (solid background)
- [ ] File size: < 1 MB
- [ ] sRGB color space

**Design Requirements:**

- [ ] Recognizable at small sizes (48×48)
- [ ] No text (illegible when small)
- [ ] High contrast (visible on any background)
- [ ] Matches brand colors (purple/cyan)
- [ ] Unique and memorable

**Testing:**

- [ ] View at 48×48 pixels (is it still recognizable?)
- [ ] View on dark background (does it stand out?)
- [ ] View on light background (does it work?)
- [ ] Compare to similar apps (is it distinctive?)

## 💡 Quick Tips

**Do:**

- ✅ Keep it simple (1-2 elements max)
- ✅ Use bold, recognizable shapes
- ✅ Ensure visibility at small sizes
- ✅ Match your brand colors
- ✅ Make it unique

**Don't:**

- ❌ Use photos (too detailed)
- ❌ Add text (illegible when small)
- ❌ Use transparency (requirement violation)
- ❌ Make it too complex (confusing)
- ❌ Copy other apps' icons
