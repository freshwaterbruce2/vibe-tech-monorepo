## ✅ Step-by-Step Workflow

### STEP 1: Screenshots (Start Here - Easiest!)

**Time:** 30 minutes

1. Read `SCREENSHOT_CAPTURE_GUIDE.md`
2. Open app (browser or phone)
3. Navigate to each view
4. Capture screenshots
5. Save to `screenshots/` with correct names
6. ✅ Check: Do you have at least 3 screenshots?

### STEP 2: App Icon

**Time:** 30-60 minutes

1. Read `ICON_DESIGN_GUIDE.md`
2. Choose: Canva template OR Figma custom
3. Design icon with purple/cyan colors
4. Export as 1024×1024 PNG (no transparency)
5. Save as `icon-1024x1024.png`
6. ✅ Check: Is it recognizable at 48×48 pixels?

### STEP 3: Feature Graphic

**Time:** 30-60 minutes

1. Read `FEATURE_GRAPHIC_GUIDE.md`
2. Choose layout (text-focused or screenshot-focused)
3. Create 1024×500 canvas
4. Add gradient background
5. Add title, tagline, elements
6. Export as `feature-graphic-1024x500.png`
7. ✅ Check: Is text readable at thumbnail size?

## 🎯 Quality Checklist

### Before You Upload to Play Store

**Screenshots:**

- [ ] At least 3 screenshots (7 recommended)
- [ ] Size: 1080×1920 pixels each
- [ ] Format: PNG
- [ ] Shows actual app content (not empty screens)
- [ ] Text is readable
- [ ] No personal information visible

**App Icon:**

- [ ] Size: Exactly 1024×1024 pixels
- [ ] Format: PNG (32-bit)
- [ ] No transparency (solid background)
- [ ] Recognizable at small sizes
- [ ] Matches brand colors

**Feature Graphic:**

- [ ] Size: Exactly 1024×500 pixels
- [ ] Format: PNG or JPEG
- [ ] App name is clear and readable
- [ ] Professional appearance
- [ ] No misleading claims

## 💡 Pro Tips

**Screenshots:**

- ✅ Capture from real device (better quality)
- ✅ Show diverse features (not all the same screen)
- ✅ Ensure good lighting/contrast
- ❌ Don't include loading spinners
- ❌ Don't show error messages

**Icon:**

- ✅ Test at 48×48 pixels (home screen size)
- ✅ High contrast (stands out)
- ✅ Simple shapes (complex = blurry when small)
- ❌ Don't use text (illegible)
- ❌ Don't use photos (too detailed)

**Feature Graphic:**

- ✅ Bold, large title
- ✅ Clear benefit ("AI Homework Manager")
- ✅ Professional, polished look
- ❌ Don't clutter with too much text
- ❌ Don't use generic stock photos

## ❓ Need Help?

**Common Issues:**

**"Screenshots are wrong size"**

- Use browser DevTools device mode (Pixel 5)
- Or crop/resize in Photopea

**"Icon looks blurry when small"**

- Simplify design (fewer elements)
- Increase contrast
- Use bold shapes

**"Feature graphic text is unreadable"**

- Make font larger (80px minimum for title)
- Increase contrast with background
- Use bold fonts
