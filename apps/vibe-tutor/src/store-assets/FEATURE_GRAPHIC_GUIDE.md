# Feature Graphic Design Guide

**Required:** 1024×500 PNG (banner for Play Store listing header)

---

## 🎨 Recommended Layout

```
┌─────────────────────────────────────────────────────────────┐
│  [Icon/Logo]    VIBE-TUTOR                    [Screenshot]  │
│                 AI Homework Manager for Teens                │
│                 ADHD & Autism Support                        │
└─────────────────────────────────────────────────────────────┘
     1024×500 pixels
```

### Layout Breakdown

**Left Third (300px):**

- App icon or logo
- Floating/glowing effect

**Center (400px):**

- "Vibe-Tutor" title (large, bold)
- Subtitle: "AI Homework Manager for Teens"
- Sub-subtitle: "ADHD & Autism Support"

**Right Third (300px):**

- Phone screenshot mockup (dashboard or AI chat)
- Or: Feature icons (brain, calendar, token)

---

## 🛠️ How to Create (Canva - Recommended)

**Step 1:** Go to <https://canva.com>

**Step 2:** Create custom size: 1024×500 px

**Step 3:** Background:

- Add gradient background
- Color 1: #0F0F23 (dark purple)
- Color 2: #1A1627 (lighter purple)
- Direction: Left to right
- Add subtle overlay texture (optional)

**Step 4:** Add Text:

```
VIBE-TUTOR
- Font: Montserrat Bold or Poppins Bold
- Size: 80-100 px
- Color: White (#FFFFFF)
- Add text shadow (purple glow)

AI Homework Manager for Teens
- Font: Montserrat Medium
- Size: 32 px
- Color: Cyan (#06FFF0) or White

ADHD & Autism Support
- Font: Montserrat Regular
- Size: 24 px
- Color: Light purple (#8B5CF6)
```

**Step 5:** Add Elements:

- Upload your app icon (left side)
- Add screenshot (right side, optional)
- Or add icons: brain, calendar, sparkles

**Step 6:** Add Glow Effects:

- Duplicate title text
- Apply blur effect
- Set to purple/cyan color
- Reduce opacity to 60%

**Step 7:** Download:

- Format: PNG
- Size: 1024×500 (verify)
- Quality: Maximum
- Save to `store-assets/feature-graphic-1024x500.png`

## 🛠️ Quick Template (Figma)

**Step 1:** Create frame 1024×500

**Step 2:** Background gradient:

```
Rectangle: 1024×500
Fill: Linear gradient
  - Color 1: #0F0F23 (left)
  - Color 2: #1A1627 (right)
  - Angle: 90°
```

**Step 3:** Title text:

```
Text: "VIBE-TUTOR"
Font: Inter Bold or Montserrat Bold
Size: 90px
Color: #FFFFFF
Position: Center-left (X: 300, Y: 150)
Effects: Drop shadow (0, 4, 20, purple, 50% opacity)
```

**Step 4:** Subtitle:

```
Text: "AI Homework Manager for Teens"
Font: Inter Medium
Size: 32px
Color: #06FFF0
Position: Below title (X: 300, Y: 250)
```

**Step 5:** Sub-subtitle:

```
Text: "ADHD & Autism Support • Focus Timer • Visual Schedules"
Font: Inter Regular
Size: 20px
Color: #8B5CF6
Position: Below subtitle (X: 300, Y: 300)
```

**Step 6:** Add icon/screenshot (optional):

- Left side: Your app icon (200×200)
- Right side: Phone mockup with screenshot
- Or: 3 feature icons (brain, calendar, tokens)

**Step 7:** Export as PNG 1024×500

## ✅ Feature Graphic Checklist

**Technical Requirements:**

- [ ] Size: Exactly 1024×500 pixels
- [ ] Format: PNG or JPEG
- [ ] No transparency required (can use solid background)
- [ ] File size: < 1 MB
- [ ] Text is readable at thumbnail size

**Design Requirements:**

- [ ] App name prominent and readable
- [ ] Tagline explains what app does
- [ ] Visually appealing and professional
- [ ] Matches brand colors (purple/cyan)
- [ ] No excessive text (keep it simple)

**Content Guidelines:**

- [ ] No misleading claims
- [ ] No pricing info (app is free)
- [ ] No URLs or contact info
- [ ] No Google Play badges
- [ ] Family-friendly imagery

## 🎯 What Makes a Good Feature Graphic?

**Great Examples:**

- Clear, readable title
- Obvious what the app does (in 2 seconds)
- Professional, polished look
- Eye-catching colors
- Simple, not cluttered

**Avoid:**

- Too much text
- Tiny, unreadable fonts
- Cluttered layout
- Stock photos that don't relate
- Generic, boring design
