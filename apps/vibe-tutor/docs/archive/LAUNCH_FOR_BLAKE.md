# 🚀 Quick Launch Guide - Blake's Vibe Tutor

## Ready to Launch in 3 Steps

### Step 1: Test the App

```bash
cd C:\dev\apps\vibe-tutor
pnpm run dev
```

Then open: <http://localhost:5173>

### Step 2: Build for Production

```bash
pnpm run build
```

### Step 3: Install on Blake's Device

#### For Computer (Web)

1. Open Chrome/Edge
2. Go to the app URL
3. Click ⋮ menu → "Install Vibe Tutor"
4. It will work offline!

#### For Android Tablet/Phone

```bash
# Build the APK
pnpm run android:full-build

# Install on device
pnpm run android:deploy
```

## 🔧 Troubleshooting

### If app doesn't load

```bash
# Clear cache and rebuild
rm -rf dist node_modules
pnpm install
pnpm run build
```

### If sounds don't work

- Check volume in Sensory Settings
- Enable sounds in device settings
- Restart the app

### If points don't save

- Check internet connection
- Clear browser cache
- Contact support
